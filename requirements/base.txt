# Midtrans vs Xendit - Pertimbangan Teknis:

# MIDTRANS
# Pros:
# - Official library Python maintained
# - Dokumentasi bahasa Indonesia lengkap
# - Fee lebih rendah untuk VA Bank lokal
# - Support semua bank major (BCA, Mandiri, BNI, BRI, Permata)
# - Settlement langsung ke rekening yayasan

# Cons:
# - Dashboard agak kompleks untuk non-technical user

# XENDIT
# Pros:
# - Dashboard lebih modern & user-friendly
# - API lebih RESTful & consistent
# - Support VA + QRIS + e-wallet

# Cons:
# - Official Python library kurang maintenance
# - Fee lebih tinggi
# - Harus build wrapper sendiri untuk beberapa fitur

# REKOMENDASI: Gunakan Midtrans untuk PPDB karena:
# 1. Fee lebih murah (penting untuk yayasan)
# 2. VA Bank paling stabil
# 3. Support resmi bagus
